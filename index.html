ProseMirror Widget Examples

Use to insert a widget and supply its params. Hover and click on the edit icon to modify.

A brief description of the widgets are shown below:

  • Image: Enter img.png for a test image.
  • Textfield, Textarea, Checkbox and Select provide form elements
  • CarryForward displays previous student answers
  • InlineMath and BlockMath use MathJax. Swipe the equations below to test:
    inline: f(x)=(x+a)(x+b)
    block: \lim_{x\to 0}{\frac{e^x-1}{2x}} \overset{\left[\frac{0}{0}\right]}{\underset{\mathrm{H}}{=}} \lim_{x\to 0}{\frac{e^x}{2}}={\frac{1}{2}}
  • Spreadsheet uses Handsontable for integration. (Currently reporting a mixed content error.)
  • IFrame embeds websites, youTube, GoogleMaps... The sites won't display because of iframe restrictions.
  • MultipleChoice, CheckList and Scale form the basis for a Test/Survey/Quiz/Exam document schema.

    Click below the title and start editing. Press Enter to provide answers.

This demo also shows a custom param handler which exhibits modal behavior and input validation.
