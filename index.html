ProseMirror Widget Examples

This editor demonstrates customizations which can be added to the ProseMirror editor including
widgets, pulldown menus and modal parameter handling and validation.
Insert a widget and adjust its settings. Hover and click on the edit icon to modify (in most cases).

Content Widgets
The following widgets can be inserted within text or as a separate block of text.
  • Image: Enter img.png for a test image.
  • CarryForward displays previous student answers
  • InlineMath and BlockMath use MathJax. Swipe the equations below to test:
    inline: f(x)=(x+a)(x+b)
    block: \lim_{x\to 0}{\frac{e^x-1}{2x}} \overset{\left[\frac{0}{0}\right]}{\underset{\mathrm{H}}{=}} \lim_{x\to 0}{\frac{e^x}{2}}={\frac{1}{2}}
  • Spreadsheet uses Handsontable for integration.
  • Website embeds websites, youTube, GoogleMaps...
Question Widgets

A Survey consists of survey questions. Insert the question, specify the parameters and type below the line to enter the question.

  • Short Answer, Essay, Scale and Selection
  • MultipleChoice, CheckList. Answers work like a bulleted list
